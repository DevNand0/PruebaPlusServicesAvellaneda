# PruebaPlusServicesAvellaneda
Prueba tecnica para ingresar a Plus Services hecha en Laravel 8 con Vue
Lista de Apis

-- recetas 
(get) localhost:8000/api/receta/get/1
(get) localhost:8000/api/receta/all/1
(post) localhost:8000/api/receta/add
(put) localhost:8000/api/receta/put/1
(delete) localhost:8000/api/receta/remove/1


-- paciente 
(get) localhost:8000/api/paciente/get/1
(get) localhost:8000/api/paciente/all
(post) localhost:8000/api/paciente/add
(put) localhost:8000/api/paciente/put/1
(delete) localhost:8000/api/paciente/remove/1
